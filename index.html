My Work

  • Instamour

    • PHP
    • API
    • Mobile
    • HTML
    • Sass
    • AWS

    Instamour is a video dating app currently on the IOS and Android marketplace. Both apps and an admin communicate through an API that is built in PHP. We have scaled the apps to a 145k+ userbase.

    • Single API endpoint used for IOS, Android, Admin, and Web applications. API is built on a custom PHP framework.
    • Full stack is hosted on AWS. Uses a basic EC2 instance running Nginx and is setup for autoscaling. An RDS/MySQL server is setup as the database.
    • Uses AWS SNS for push notifications.
    • All videos are rendered through FFMpeg and stored on S3.
    • Uses Facebook Oath for user login.
    • A Facebook canvas app was setup around the web app for easy Facebook integration and user aquisition.

  • CHTSHT.io

    • PHP
    • Phalcon
    • MySQL
    • Redis
    • API
    • Mobile
    • HTML
    • Sass
    • AWS

    CHTSHT.io is a free, open source web app that allows users to post wiki style lists, or Cheetsheets, for various linux commands

    • API is built in PHP using the Phalcon framework. MySQL is used for the database. Redis is used for caching. Hosted on an EC2 instance running Nginx.
    • User authentication is done through Github Oath. Only tokens are stored in the database.
    • The web frontend uses javascript/JQuery and Html5 elements to create a seamless, single page application for creating and editing CHTSHTs.
    • All code can be viewed on Github: github.com/chtsht
